# Day 15 Notes

## What is Data Management?

Data Management involves:

- Collecting data
- Storing data
- Updating data
- Importing data
- Exporting data
- Maintaining data quality

Good data management improves business reliability.

---

## What is Data Loader?

Data Loader is a Salesforce tool used for:

- Insert records
- Update records
- Delete records
- Export records
- Perform bulk operations

Useful for handling thousands of records efficiently.

---

## Data Migration

Data Migration means moving data from one system to another.

Example:

Excel Sheets
↓
Salesforce CRM

Challenges:

- Duplicate data
- Missing values
- Format mismatches
- Invalid records

---

## Data Quality

Good Data Characteristics:

- Accurate
- Complete
- Consistent
- Reliable
- Up-to-date

Poor data creates business problems.

---

## Duplicate Prevention

Salesforce provides:

- Validation Rules
- Duplicate Rules
- Matching Rules
- Required Fields

These help maintain clean data.

---

## Enterprise Impact of Bad Data

Bad data may cause:

- Wrong reports
- Incorrect notifications
- Financial mistakes
- Compliance risks
- Poor business decisions

---

## Key Learning

Data is one of the most valuable assets in enterprise systems. Maintaining high-quality data is essential for reliable automation, reporting, and decision-making.
