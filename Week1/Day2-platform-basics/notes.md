## Important Concepts

### App
A collection of objects, tabs, and tools for specific business purposes.

### Object
A database table used to store information.

### Tab
UI element used to access objects.

### Configuration
Customization without coding.

### Coding
Advanced customization using Apex.

### Multi-Tenant Architecture
Multiple companies share the same infrastructure securely.
