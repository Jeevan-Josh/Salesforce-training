# Salesforce Summer Program 2026

## Overview

This repository contains my complete learning journey through the Salesforce Summer Program.

Throughout this program, I explored Salesforce fundamentals, CRM concepts, automation, Apex development, Lightning Web Components (LWC), testing, DevOps workflows, AI-powered solutions, and enterprise application architecture.

The project work throughout the program was based on a College Management System, helping me understand how real-world Salesforce applications are designed, automated, tested, and maintained.
